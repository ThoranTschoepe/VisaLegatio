# VisaLegatio Frontend

## Quick Start for Hackathon

### 1. Setup
```bash
# Navigate to backend folder
cd frontend

# Install dependencies
npm install

npm run dev
```

### 2. The Website will be available at:
- **Applicant Site**: http://localhost:3000
- **Embassy Site**: http://localhost:3000/embassy
